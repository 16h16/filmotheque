## Filmotheque

**Filmotheque** is a Laravel application that organizes a library of movies

## Features

**Movie** 
- Display
- Create
- Update
- Delete

**Category**
- Display
- Create
- Update
- Delete

**Actor**
- Display
- Create
- Update
- Delete
